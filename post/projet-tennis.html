Tennis project

Context

The CAP Tennis Club of PÃ©rigueux has decided to acquire an application for the management of tennis matches. It will be used during the matches by the referees to update the score. The objective is to realize a web application assuming that the referees will have at their disposal a tablet connected to the internet. This website will be developed in PHP and HTML/CSS and will use the website used in block 1 as well as its MVC architecture.

Specifications for the tennis project.

  • Website: The website will be based on one of the websites seen in block 1 and its MVC architecture. Adapt the graphical interface, thus the CSS file, to the context: modify in particular the logo, the menus, and provide an ergonomic interface for the forms.
  • Initialization form: check that the game can only start with complete and valid information.
  • Points entry form: The page should display the requested information in an ergonomic manner for the referee.
